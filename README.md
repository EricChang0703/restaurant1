# Getting Started

**Clone respository to your local computer**

```
$ git clone https://github.com/EricChang0703/Restaurant
```

**Install by npm**

```
$ npm install
```

**Run**

```
npm run start
```

**Terminal show the message**

```
Express is running on localhost:3000
```

**Now you can browse the website on**

```
http://localhost:3000
```

**Close**

```
ctrl + c
```
